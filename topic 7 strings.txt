What is a String?
-----------------
A String is a sequence of characters. In Java, String is an object that
represents a sequence of characters. It is one of the most commonly used
data types in Java programming and automation testing.

Key Characteristics:
--------------------
- String is a class in java.lang package
- Strings are immutable (cannot be changed once created)
- Strings are stored in String Pool for memory optimization
- String can be created using literal or new keyword


Why Important for Automation?
-----------------------------
- Store and validate page titles
- Handle URLs
- Process text from web elements
- Validate error messages
- Handle API request/response data
- Work with XPath and CSS selectors
- Process test data
- Generate dynamic test data
- Log messages and reports


================================================================================
                    1. CREATING STRINGS
================================================================================

There are two ways to create a String:

1. String Literal (Recommended)
2. Using new keyword


String Pool Concept:
--------------------
When you create a String literal, Java checks the String Pool.
If the same string exists, it returns reference to existing string.
If not, it creates new string in the pool.

+--------------------------------------------------+
|                    HEAP MEMORY                    |
|  +--------------------------------------------+  |
|  |              STRING POOL                   |  |
|  |                                            |  |
|  |    +--------+    +--------+    +--------+  |  |
|  |    |"Hello" |    |"World" |    |"Java"  |  |  |
|  |    +--------+    +--------+    +--------+  |  |
|  |        ^             ^                     |  |
|  |        |             |                     |  |
|  +--------------------------------------------+  |
|           |             |                        |
|        s1, s2          s3                        |
|                                                  |
|  +--------------------------------------------+  |
|  |           OUTSIDE STRING POOL              |  |
|  |                                            |  |
|  |    +--------+                              |  |
|  |    |"Hello" |  <-- s4 (created with new)   |  |
|  |    +--------+                              |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+

EXAMPLE 1: Creating Strings - Different Methods
--------------------------------------------------------------------------------

public class StringCreation {
    public static void main(String[] args) {
        
        // Method 1: String Literal (Recommended)
        System.out.println("=== Method 1: String Literal ===");
        String s1 = "Hello";
        String s2 = "Hello";
        String s3 = "World";
        
        System.out.println("s1 = " + s1);
        System.out.println("s2 = " + s2);
        System.out.println("s3 = " + s3);
        
        // s1 and s2 point to same object in String Pool
        System.out.println("s1 == s2: " + (s1 == s2));    // true
        
        System.out.println("");
        
        // Method 2: Using new keyword
        System.out.println("=== Method 2: Using new Keyword ===");
        String s4 = new String("Hello");
        String s5 = new String("Hello");
        
        System.out.println("s4 = " + s4);
        System.out.println("s5 = " + s5);
        
        // s4 and s5 are different objects (outside String Pool)
        System.out.println("s4 == s5: " + (s4 == s5));    // false
        System.out.println("s4.equals(s5): " + s4.equals(s5));    // true
        
        System.out.println("");
        
        // Method 3: From character array
        System.out.println("=== Method 3: From Character Array ===");
        char[] charArray = {'J', 'a', 'v', 'a'};
        String s6 = new String(charArray);
        System.out.println("s6 = " + s6);
        
        System.out.println("");
        
        // Method 4: Empty and blank strings
        System.out.println("=== Method 4: Empty and Blank Strings ===");
        String empty = "";
        String blank = "   ";
        String nullString = null;
        
        System.out.println("Empty string length: " + empty.length());
        System.out.println("Blank string length: " + blank.length());
        System.out.println("Null string: " + nullString);
    }
}

================================================================================
                    2. STRING IMMUTABILITY
================================================================================

Strings in Java are immutable - once created, they cannot be changed.
Any operation that appears to modify a String actually creates a new String.


--------------------------------------------------------------------------------
EXAMPLE 2: String Immutability Demonstration
--------------------------------------------------------------------------------

public class StringImmutability {
    public static void main(String[] args) {
        
        String original = "Hello";
        System.out.println("Original string: " + original);
        System.out.println("Original hashCode: " + original.hashCode());
        
        // This creates a NEW string, doesn't modify original
        String modified = original.concat(" World");
        
        System.out.println("");
        System.out.println("After concat operation:");
        System.out.println("Original string: " + original);    // Still "Hello"
        System.out.println("Original hashCode: " + original.hashCode());
        System.out.println("Modified string: " + modified);    // "Hello World"
        System.out.println("Modified hashCode: " + modified.hashCode());
        
        System.out.println("");
        
        // Another example
        String s = "Java";
        System.out.println("Before toUpperCase: " + s);
        
        String upper = s.toUpperCase();
        
        System.out.println("After toUpperCase:");
        System.out.println("s = " + s);           // Still "Java"
        System.out.println("upper = " + upper);   // "JAVA"
        
        System.out.println("");
        
        // To actually change the value, reassign
        System.out.println("=== Reassigning to Change Value ===");
        String name = "John";
        System.out.println("Before: " + name);
        
        name = name + " Doe";    // Creates new string, assigns to same variable
        System.out.println("After: " + name);
    }
}

+--------+      +--------------+

+--------+      +--------------+
|"Hello" |      |"Hello World" | <-- s now points here
+--------+      +--------------+

public class StringCharacterAccess {
    public static void main(String[] args) {
        
        String text = "Selenium";
        
        // charAt(index) - Returns character at specified index
        System.out.println("=== charAt() Method ===");
        System.out.println("String: " + text);
        System.out.println("Index:  01234567");
        System.out.println("");
        
        System.out.println("charAt(0) = " + text.charAt(0));    // S
        System.out.println("charAt(3) = " + text.charAt(3));    // e
        System.out.println("charAt(7) = " + text.charAt(7));    // m
        
        // Get last character
        System.out.println("Last char = " + text.charAt(text.length() - 1));
        
        System.out.println("");
        
        // toCharArray() - Converts string to character array
        System.out.println("=== toCharArray() Method ===");
        char[] chars = text.toCharArray();
        
        System.out.println("Character array:");
        for (int i = 0; i < chars.length; i++) {
            System.out.println("chars[" + i + "] = " + chars[i]);
        }
        
        System.out.println("");
        
        // Practical use: Count specific characters
        System.out.println("=== Practical: Count Vowels ===");
        String sentence = "Hello World";
        int vowelCount = 0;
        
        for (char c : sentence.toLowerCase().toCharArray()) {
            if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') {
                vowelCount++;
            }
        }
        System.out.println("Vowels in \"" + sentence + "\": " + vowelCount);
    }
}

public class StringComparisonMethods {
    public static void main(String[] args) {
        
        String s1 = "Hello";
        String s2 = "Hello";
        String s3 = "hello";
        String s4 = new String("Hello");
        String s5 = "World";
        String s6 = "Apple";
        
        // equals() - Compares content (case-sensitive)
        System.out.println("=== equals() Method ===");
        System.out.println("\"Hello\".equals(\"Hello\") = " + s1.equals(s2));
        System.out.println("\"Hello\".equals(\"hello\") = " + s1.equals(s3));
        System.out.println("\"Hello\".equals(new String(\"Hello\")) = " + s1.equals(s4));
        
        System.out.println("");
        
        // equalsIgnoreCase() - Compares content (case-insensitive)
        System.out.println("=== equalsIgnoreCase() Method ===");
        System.out.println("\"Hello\".equalsIgnoreCase(\"hello\") = " + s1.equalsIgnoreCase(s3));
        System.out.println("\"Hello\".equalsIgnoreCase(\"HELLO\") = " + s1.equalsIgnoreCase("HELLO"));
        System.out.println("\"Hello\".equalsIgnoreCase(\"World\") = " + s1.equalsIgnoreCase(s5));
        
        System.out.println("");
        
        // == vs equals()
        System.out.println("=== == vs equals() ===");
        System.out.println("s1 == s2 (both literals): " + (s1 == s2));
        System.out.println("s1 == s4 (literal vs new): " + (s1 == s4));
        System.out.println("s1.equals(s4): " + s1.equals(s4));
        
        System.out.println("");
        
        // compareTo() - Lexicographical comparison
        // Returns: negative if s1 < s2, 0 if equal, positive if s1 > s2
        System.out.println("=== compareTo() Method ===");
        System.out.println("\"Hello\".compareTo(\"Hello\") = " + s1.compareTo(s2));
        System.out.println("\"Hello\".compareTo(\"World\") = " + s1.compareTo(s5));
        System.out.println("\"Hello\".compareTo(\"Apple\") = " + s1.compareTo(s6));
        System.out.println("\"Apple\".compareTo(\"Hello\") = " + s6.compareTo(s1));
        
        System.out.println("");
        
        // Explanation of compareTo
        System.out.println("=== compareTo() Explanation ===");
        System.out.println("Negative: First string comes before second alphabetically");
        System.out.println("Zero: Strings are equal");
        System.out.println("Positive: First string comes after second alphabetically");
        
        System.out.println("");
        
        // compareToIgnoreCase()
        System.out.println("=== compareToIgnoreCase() Method ===");
        System.out.println("\"Hello\".compareToIgnoreCase(\"hello\") = " + s1.compareToIgnoreCase(s3));
        System.out.println("\"Hello\".compareToIgnoreCase(\"WORLD\") = " + s1.compareToIgnoreCase("WORLD"));
    }
}

public class StringCaseConversion {
    public static void main(String[] args) {

        // toLowerCase() - Converts all characters to lowercase

        System.out.println("=== toUpperCase() Method ===");
        System.out.println("\"" + mixed + "\".toUpperCase() = " + mixed.toUpperCase());

        System.out.println("");
        
        // Original string is unchanged (immutability)
        System.out.println("=== Original Unchanged ===");
        System.out.println("Original: " + mixed);
        
        System.out.println("");
        
        // Practical: Case-insensitive comparison
        System.out.println("=== Practical: Case-Insensitive Search ===");
        String pageTitle = "Welcome to Google Search";
        String searchText = "GOOGLE";
        
        boolean found = pageTitle.toLowerCase().contains(searchText.toLowerCase());
        System.out.println("Page Title: " + pageTitle);
        System.out.println("Search for: " + searchText);
        System.out.println("Found (case-insensitive): " + found);
        
        System.out.println("");
        
        // Practical: Convert to Title Case
        System.out.println("=== Practical: Title Case Conversion ===");
        String input = "john doe";
        String[] words = input.split(" ");
        StringBuilder titleCase = new StringBuilder();
        
        for (String word : words) {
            if (word.length() > 0) {
                titleCase.append(word.substring(0, 1).toUpperCase());
                titleCase.append(word.substring(1).toLowerCase());
                titleCase.append(" ");
            }
        }
        
        System.out.println("Input: " + input);
        System.out.println("Title Case: " + titleCase.toString().trim());
    }
}

public class StringTrimMethods {
    public static void main(String[] args) {
        
        String text = "   Hello World   ";
        String leadingSpace = "   Hello";
        String trailingSpace = "Hello   ";
        
        System.out.println("=== Original Strings ===");
        System.out.println("text: \"" + text + "\" (length: " + text.length() + ")");
        
        System.out.println("");
        
        // trim() - Removes leading and trailing whitespace (ASCII spaces)
        System.out.println("=== trim() Method ===");
        String trimmed = text.trim();
        System.out.println("trim(): \"" + trimmed + "\" (length: " + trimmed.length() + ")");
        
        System.out.println("");
        
        // strip() - Removes leading and trailing whitespace (Unicode-aware) - Java 11+
        System.out.println("=== strip() Method (Java 11+) ===");
        String stripped = text.strip();
        System.out.println("strip(): \"" + stripped + "\" (length: " + stripped.length() + ")");
        
        System.out.println("");
        
        // stripLeading() - Removes only leading whitespace - Java 11+
        System.out.println("=== stripLeading() Method (Java 11+) ===");
        String stripLead = text.stripLeading();
        System.out.println("stripLeading(): \"" + stripLead + "\" (length: " + stripLead.length() + ")");
        
        System.out.println("");
        
        // stripTrailing() - Removes only trailing whitespace - Java 11+
        System.out.println("=== stripTrailing() Method (Java 11+) ===");
        String stripTrail = text.stripTrailing();
        System.out.println("stripTrailing(): \"" + stripTrail + "\" (length: " + stripTrail.length() + ")");
        
        System.out.println("");
        
        // trim() vs strip() difference
        System.out.println("=== trim() vs strip() ===");
        System.out.println("trim() - Removes ASCII whitespace (space, tab, newline)");
        System.out.println("strip() - Removes Unicode whitespace (more comprehensive)");
        System.out.println("For most cases, both work the same.");
        
        System.out.println("");
        
        // Practical: Clean user input
        System.out.println("=== Practical: Clean User Input ===");
        String userInput = "  admin@email.com  ";
        String cleanInput = userInput.trim().toLowerCase();

public class StringReplaceMethods {
    public static void main(String[] args) {

        // replace(oldString, newString) - Replace all occurrences of string
        System.out.println("=== replace(String, String) ===");

        System.out.println("replace(\"o\", \"00\"): " + text.replace("o", "00"));

        // replaceAll(regex, replacement) - Replace using regular expression
        System.out.println("=== replaceAll(regex, replacement) ===");

        System.out.println("Original: " + numbers);
        System.out.println("Remove all digits: " + numbers.replaceAll("[0-9]", "X"));

        String multiSpace = "Hello    World   Java";
        System.out.println("Original: \"" + multiSpace + "\"");
        System.out.println("Single space: \"" + multiSpace.replaceAll("\\s+", " ") + "\"");
        
        System.out.println("");
        
        // replaceFirst(regex, replacement) - Replace only first occurrence
        System.out.println("=== replaceFirst(regex, replacement) ===");
        System.out.println("Original: " + text);
        System.out.println("replaceFirst(\"World\", \"Java\"): " + text.replaceFirst("World", "Java"));
        
        System.out.println("");
        
        // Practical examples
        System.out.println("=== Practical Examples ===");

        System.out.println("Masked: " + masked);
        
        System.out.println("");
        
        // Clean phone number
        String phone = "(123) 456-7890";
        String cleanPhone = phone.replaceAll("[^0-9]", "");
        System.out.println("Original: " + phone);
        System.out.println("Clean: " + cleanPhone);

public class StringStartEndMethods {
    public static void main(String[] args) {

        System.out.println("startsWith(\"http\"): " + url.startsWith("http"));
        System.out.println("startsWith(\"ftp\"): " + url.startsWith("ftp"));
        
        // startsWith with offset
        System.out.println("startsWith(\"www\", 8): " + url.startsWith("www", 8));
        
        System.out.println("");
        
        // endsWith() - Check if string ends with specified suffix
        System.out.println("=== endsWith() Method ===");
        System.out.println("Filename: " + filename);
        System.out.println("endsWith(\".pdf\"): " + filename.endsWith(".pdf"));
        System.out.println("endsWith(\".doc\"): " + filename.endsWith(".doc"));
        System.out.println("endsWith(\".PDF\"): " + filename.endsWith(".PDF"));
        
        System.out.println("");
        
        // Practical: Validate URL protocol
        System.out.println("=== Practical: Validate URL Protocol ===");
        String[] urls = {
            "https://secure.com",
            "http://insecure.com",
            "ftp://files.com",
            "www.noprotocol.com"
        };
        
        for (String u : urls) {
            String protocol;
            if (u.startsWith("https://")) {
                protocol = "HTTPS (Secure)";
            } else if (u.startsWith("http://")) {
                protocol = "HTTP (Insecure)";
            } else if (u.startsWith("ftp://")) {
                protocol = "FTP";
            } else {
                protocol = "Unknown/Missing";
            }
            System.out.println(u + " -> " + protocol);
        }
        
        System.out.println("");
        
        // Practical: Validate file types
        System.out.println("=== Practical: Validate File Types ===");
        String[] files = {"report.pdf", "image.png", "data.xlsx", "script.js"};
        String[] allowedExtensions = {".pdf", ".xlsx", ".doc"};
        
        for (String file : files) {
            boolean allowed = false;
            for (String ext : allowedExtensions) {
                if (file.endsWith(ext)) {
                    allowed = true;
                    break;
                }
            }
            System.out.println(file + " -> " + (allowed ? "ALLOWED" : "NOT ALLOWED"));
        }
        
        System.out.println("");
        
        // Practical: Email validation
        System.out.println("=== Practical: Basic Email Validation ===");
        String[] emails = {"user@email.com", "invalid.email", "test@", "@domain.com"};
        
        for (String e : emails) {
            boolean valid = e.contains("@") && 
                           !e.startsWith("@") && 
                           !e.endsWith("@") &&
                           e.indexOf("@") == e.lastIndexOf("@");
            System.out.println(e + " -> " + (valid ? "VALID" : "INVALID"));
        }
    }
}

        int number = 42;

        // Method 3: StringBuilder (efficient for multiple concatenations)
        System.out.println("=== StringBuilder ===");
        StringBuilder sb = new StringBuilder();
        sb.append("Hello");
        sb.append(" ");
        sb.append("World");
        sb.append(" ");
        sb.append(2024);
        String result4 = sb.toString();
        System.out.println("StringBuilder result: " + result4);
        
        System.out.println("");
        
        // Performance comparison
        System.out.println("=== Performance Comparison ===");
        
        // Using + in loop (INEFFICIENT)
        long start1 = System.currentTimeMillis();
        String inefficient = "";
        for (int i = 0; i < 10000; i++) {
            inefficient = inefficient + "a";    // Creates new String each time!
        }
        long end1 = System.currentTimeMillis();
        System.out.println("Using + operator (10000 iterations): " + (end1 - start1) + " ms");
        
        // Using StringBuilder (EFFICIENT)
        long start2 = System.currentTimeMillis();
        StringBuilder efficient = new StringBuilder();
        for (int i = 0; i < 10000; i++) {
            efficient.append("a");    // Modifies same object
        }
        String efficientResult = efficient.toString();
        long end2 = System.currentTimeMillis();
        System.out.println("Using StringBuilder (10000 iterations): " + (end2 - start2) + " ms");
        
        System.out.println("");
        
        // Method 4: String.format()
        System.out.println("=== String.format() ===");
        String name = "John";
        int age = 25;

public class StringAutomationValidation {
    public static void main(String[] args) {
        
        System.out.println("=== Input Validation Examples ===");
        System.out.println("");
        
        // Email validation
        System.out.println("--- Email Validation ---");
        String[] emails = {
            "user@example.com",
            "invalid.email",
            "user@.com",
            "@example.com",
            "user@example",
            "user.name@example.co.uk"
        };
        
        for (String email : emails) {
            boolean valid = isValidEmail(email);
            System.out.println(email + " -> " + (valid ? "VALID" : "INVALID"));
        }
        
        System.out.println("");
        
        // Phone number validation
        System.out.println("--- Phone Number Validation ---");
        String[] phones = {
            "1234567890",
            "123-456-7890",
            "(123) 456-7890",
            "12345",
            "abcdefghij"
        };
        
        for (String phone : phones) {
            String cleaned = phone.replaceAll("[^0-9]", "");
            boolean valid = cleaned.length() == 10;
            System.out.println(phone + " -> " + (valid ? "VALID" : "INVALID") + 
                             " (digits: " + cleaned + ")");
        }
        
        System.out.println("");
        
        // Password strength validation
        System.out.println("--- Password Strength Validation ---");
        String[] passwords = {
            "abc",
            "abcdefgh",
            "Abcdefgh",
            "Abcdefg1",
            "Abcdef1!"
        };
        
        for (String password : passwords) {
            String strength = checkPasswordStrength(password);
            System.out.println(password + " -> " + strength);
        }
        
        System.out.println("");
        
        // Username validation
        System.out.println("--- Username Validation ---");
        String[] usernames = {
            "john_doe",
            "john.doe",
            "john-doe",
            "jo",
            "john doe",
            "john@doe"
        };
        
        for (String username : usernames) {
            boolean valid = isValidUsername(username);
            System.out.println(username + " -> " + (valid ? "VALID" : "INVALID"));
        }
    }
    
    public static boolean isValidEmail(String email) {
        if (email == null || email.isEmpty()) {
            return false;
        }
        
        // Basic validation
        boolean hasAtSymbol = email.contains("@");
        boolean hasDot = email.contains(".");
        boolean notStartWithAt = !email.startsWith("@");
        boolean notEndWithAt = !email.endsWith("@");
        boolean notEndWithDot = !email.endsWith(".");
        boolean singleAt = email.indexOf("@") == email.lastIndexOf("@");
        boolean atBeforeDot = email.indexOf("@") < email.lastIndexOf(".");
        
        return hasAtSymbol && hasDot && notStartWithAt && notEndWithAt && 
               notEndWithDot && singleAt && atBeforeDot;
    }
    
    public static String checkPasswordStrength(String password) {
        if (password.length() < 6) {
            return "WEAK (too short)";
        }
        
        boolean hasUpper = !password.equals(password.toLowerCase());
        boolean hasLower = !password.equals(password.toUpperCase());
        boolean hasDigit = password.matches(".*\\d.*");
        boolean hasSpecial = password.matches(".*[!@#$%^&*()_+\\-=\\[\\]{};':\"\\\\|,.<>\\/?].*");
        
        int score = 0;
        if (hasUpper) score++;
        if (hasLower) score++;
        if (hasDigit) score++;
        if (hasSpecial) score++;
        
        if (score <= 1) return "WEAK";
        if (score == 2) return "MEDIUM";
        if (score == 3) return "STRONG";
        return "VERY STRONG";
    }
    
    public static boolean isValidUsername(String username) {
        if (username == null || username.length() < 3) {
            return false;
        }
        
        // Only allow alphanumeric, underscore, and dot
        return username.matches("^[a-zA-Z0-9_.]+$");
    }

    // Simple JSON value extractor (for demonstration)
    public static String extractJsonValue(String json, String key) {
        String searchKey = "\"" + key + "\":";
        int keyIndex = json.indexOf(searchKey);
        
        if (keyIndex == -1) {
            return null;
        }
        
        int valueStart = keyIndex + searchKey.length();
        char firstChar = json.charAt(valueStart);
        
        if (firstChar == '"') {
            // String value
            int valueEnd = json.indexOf("\"", valueStart + 1);
            return json.substring(valueStart + 1, valueEnd);
        } else if (firstChar == '{') {
            // Object value - skip for simplicity
            return "[object]";
        } else {
            // Number or boolean
            int valueEnd = valueStart;
            while (valueEnd < json.length() && 
                   json.charAt(valueEnd) != ',' && 
                   json.charAt(valueEnd) != '}') {
                valueEnd++;
            }
            return json.substring(valueStart, valueEnd);
        }
    }
    
    // Simple XML value extractor (for demonstration)
    public static String extractXmlValue(String xml, String tag) {
        String openTag = "<" + tag + ">";
        String closeTag = "</" + tag + ">";
        
        int startIndex = xml.indexOf(openTag);
        int endIndex = xml.indexOf(closeTag);
        
        if (startIndex == -1 || endIndex == -1) {
            return null;
        }
        
        return xml.substring(startIndex + openTag.length(), endIndex);
    }

        if (str == null || str.trim().isEmpty()) {

        } catch (NumberFormatException e) {
            return null;
        }
    }

public class StringFormatFixed {
    public static void main(String[] args) {
        
        // Correct number of arguments
        String result1 = String.format("Name: %s, Age: %d, City: %s", "John", 25, "NYC");
        System.out.println(result1);
        
        // Correct types
        String result2 = String.format("Age: %d", 25);
        System.out.println(result2);
        
        // Use %s for any type (safest)
        String result3 = String.format("Value: %s", 25);
        System.out.println(result3);
        
        // Safe formatting method
        System.out.println("");
        System.out.println("=== Safe Formatting ===");
        System.out.println(safeFormat("Hello %s", "World"));
        System.out.println(safeFormat("Number: %d", 42));
        System.out.println(safeFormat("Too few: %s %s", "only one"));
    }
    
    public static String safeFormat(String format, Object... args) {
        try {
            return String.format(format, args);
        } catch (Exception e) {
            return "Format error: " + e.getMessage();
        }
    }

Not admin

FIX:
----
public class StringWhitespaceFixed {
    public static void main(String[] args) {
        
        String expected = "Success";
        String actual = "Success ";
        
        // Always trim before comparison
        if (expected.trim().equals(actual.trim())) {
            System.out.println("Match (after trim)!");
        }
        
        String input = "  admin  ";
        if (input.trim().equals("admin")) {
            System.out.println("Admin user (after trim)");
        }
        
        // Best practice: Trim and normalize
        String userInput = "  ADMIN  ";
        String normalized = userInput.trim().toLowerCase();
        
        if (normalized.equals("admin")) {
            System.out.println("Admin user (normalized)");
        }
        
        // Debug whitespace issues
        System.out.println("");
        System.out.println("=== Debugging Whitespace ===");
        String problematic = "Test ";
        System.out.println("Value: '" + problematic + "'");
        System.out.println("Length: " + problematic.length());
        System.out.println("After trim length: " + problematic.trim().length());
    }
}

BEST PRACTICES:
---------------
1. Always use .equals() for String comparison, never ==
2. Put literal first in equals(): "expected".equals(variable)
3. Always check for null before calling String methods
4. Use trim() to clean user input
5. Use StringBuilder for loop concatenation
6. Use equalsIgnoreCase() for case-insensitive comparison
7. Remember String is immutable - assign results back
8. Escape special characters in split() regex
9. Check indexOf() result before using in substring()
10. Use String.format() for complex string building
