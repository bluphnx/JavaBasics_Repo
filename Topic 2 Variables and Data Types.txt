public class IntegerTypes {
    public static void main(String[] args) {
        
        byte smallNumber = 100;
        short mediumNumber = 5000;
        int regularNumber = 100000;
        long bigNumber = 9999999999L;  // Note: L at the end
        
        System.out.println("byte: " + smallNumber);
        System.out.println("short: " + mediumNumber);
        System.out.println("int: " + regularNumber);
        System.out.println("long: " + bigNumber);
    }
}

Non-Primitive Type: String

String stores text (sequence of characters).

public class StringType {
    public static void main(String[] args) {
        
        String name = "John";
        String url = "https://google.com";
        String testCaseName = "Verify Login Functionality";
        
        System.out.println("Name: " + name);
        System.out.println("URL: " + url);
        System.out.println("Test Case: " + testCaseName);
    }
}

Output:

Test Case: Verify Login Functionality
